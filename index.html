Auto Speech

About Project:

Type your english text and click play button to auto speech.
This project was build and designed with HTML - CSS and programing language JavaScript.

    Help Manu

  • play: play auto speech text - support only english language.
  • clear: delete all text.
  • Mode: enable full screen mode.
  • help : help manu for all users.

    • Shortcuts

    > FireFox Browser:
  • [ play ] = alt + shift + r
  • [ clear ] = alt + shift + c
  • [ help ] = alt + shift + q

    • > Chrome Browser:
  • [ play ] = alt + r
  • [ clear ] = alt + c
  • [ help ] = alt + q
